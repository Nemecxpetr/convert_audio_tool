# Contributing Guide

## Development setup
1. Clone the repository
2. Create a virtual environment  
3. Install in editable mode:  
   `pip install -e .`

## Coding guidelines
- Follow PEP8.
- Add type hints where possible.
- Add docstrings to all functions.

## Commit messages
Use clear messages:
- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation changes

## Pull Requests
- Keep PRs small and focused.
- Include before/after explanation if relevant.
